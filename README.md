# Aircraft Maintenance Data Analysis

A simple Python project that reads aircraft maintenance data from a CSV file and performs basic statistical analysis on flight hours using the Pandas library.

## Features

- Reads maintenance records from a CSV file
- Calculates average flight hours
- Finds maximum flight hours
- Finds minimum flight hours
- Simple and beginner-friendly code structure

## Technologies Used

- Python
- Pandas
